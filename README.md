# 별이 밥 줬어?

반려동물 배식 여부 공유 앱

___
## 제작 의도
### 테스트
1. Firebase RealtimeDatabase 활용
2. AlarmManager, 리시버 이용한 스케줄링
___

## 기능

### 동물 추가
  + 동물 종류
  + 동물 이름
### 설정
  + 초기화 설정 (개발자)
  + 알림 설정
### 배식여부 기록
  + 체크박스를 이용한 배식여부 기록
### 초기화
  + 00시마다 배식 여부 초기화
### 알림
  1. 오전 8시에 '아침' 배식 여부 체크되지 않은 동물이 존재하는 경우
  2. 오후 8시에 '저녁' 배식 여부 체크되지 않은 동물이 존재하는 경우
